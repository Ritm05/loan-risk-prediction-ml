🚀 Loan Default Prediction System

An AI-powered Loan Risk Prediction System that helps financial institutions identify potential loan defaulters using Machine Learning.
The project analyzes applicant financial data and predicts the probability of loan default through an interactive web application built with Streamlit.

📌 Project Overview

Credit risk assessment is one of the most critical tasks for banks and lending institutions. Poor credit decisions can lead to major financial losses.

This project builds a Machine Learning powered decision-support system that:

✅ Predicts whether a borrower is likely to default on a loan
✅ Compares multiple ML models to find the best performing one
✅ Provides an interactive web interface for real-time predictions
✅ Helps reduce credit loss through data-driven insights

🧠 Machine Learning Models Used

The project evaluates several ML algorithms to determine the best predictor:

Logistic Regression

Decision Tree

Random Forest

K-Nearest Neighbors

Support Vector Machine

Bernoulli Naive Bayes

Gradient Boosting

Each model is trained using a consistent pipeline and evaluated using accuracy and performance metrics.
